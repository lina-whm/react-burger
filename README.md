# Stellar Burgers

</div>

Проект разработан в рамках обучения в Яндекс Практикуме. Это приложение для заказа космических бургеров с использованием современных веб-технологий.

## 🚀 Основные возможности

- **Конструктор бургера**: Собирай свой идеальный бургер из различных ингредиентов.
- **Лента заказов**: Следи за выполнением заказов в реальном времени через WebSocket.
- **Личный кабинет**: Управляй своими персональными данными и просматривай историю заказов.
- **Адаптивный дизайн**: Приложение корректно отображается на десктопах, планшетах и мобильных устройствах.

## 🛠 Используемые технологии

- **React** — библиотека для создания пользовательских интерфейсов.
- **Redux Toolkit** — управление состоянием приложения.
- **React Router** — навигация между страницами.
- **WebSocket** — получение данных о заказах в реальном времени.
- **CSS Modules** — модульная стилизация компонентов.
- **Jest & React Testing Library** — тестирование компонентов и логики.

## 📦 Установка и запуск

1. Клонируй репозиторий:
   ```bash
   git clone <ссылка_на_репозиторий>
   cd stellar-burgers
   ```

2. Установи зависимости:
   ```bash
   npm install
   ```

3. Запусти проект в режиме разработки:
   ```bash
   npm start
   ```

4. Открой [http://localhost:3000](http://localhost:3000) в браузере.

## 🧪 Тестирование

Для запуска тестов используй команду:
```bash
npm test
```
